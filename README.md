# 📊 qa-dashboard

# 📌 Overview
The QA Dashboard is a web application built using React that provides a centralized platform to track and visualize testing reports.

It helps teams monitor Manual Testing and Automation Testing results for both UI and API testing with interactive charts and dashboards.

# 🛠 Tech Stack
⚛️ React

🎨 HTML5 & CSS3

📊 Chart Libraries (Chart.js / Recharts / ApexCharts)

🔗 REST APIs

# 🚀 Installation
1. Clone the repository

```
git clone https://github.com/your-username/your-repository-name.git
```
2. Go to project folder

```
cd your-repository-name
```
3. Install dependencies

```
npm install
```
4. Run the project

```
npm start
```
Application runs at:

```
http://localhost:3000
```
